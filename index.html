Rubber Ducky 2023 - 2026

Why? Convicted LLMs were the new transistors, I wanted to build the future computer and software experience.

How? Local server powers CLI and UI. Ground-up framework-free build. Support all models. Stream as many data sources in and out as possible (read/write fs, browser, APIs, ...). Push limits of automation.

Impact: Organically adopted by 2,000+ engineers across 40+ Microsoft orgs. Became officially recommended tooling alongside GitHub Copilot and Claude Code.

Technologies: Docker, Node.js, Express, PostgreSQL, Neo4j, sqlite, Anthropic AI SDK / API, Azure, OpenAI, Ollama, React, Cytoscape.js, Webpack, yargs, pkg, and many others.

Open UI 2019 - 2020

Why? Wanted to remove the need to create UI frameworks by downleveling UI design system needs into the platform.

How? W3C Community Group, cross enterprise. Analyze top UI design systems and frameworks, synthesize their needs into platform specs and ship in browsers. Members from Google, Apple, Salesforce, Mozilla, Microsoft, Shopify, Adobe, and more.

Impact: After I scaffolded the system and process, immediate success with memberships and adoption. Ongoing specifications and browser updates continue. Grew to lead efforts in WHATWG and CSS WG.

Technologies: W3C, WHATWG, CSS WG, HTML, CSS, and many others.

About

Hi, I'm Levi. I'm a second generation software engineer, a confessed problem-solving addict, and I prefer to tackle the so-called impossible.

I've spent 14 years in software engineering and 16 years building and leading teams. I've founded a W3C community group, shaped web standards in WHATWG and CSS WG, architected design systems powering Microsoft's biggest products, and built agentic developer tools adopted by thousands of engineers.

I operate at the intersection of deep technical work and organizational leadership. I can go from writing the code to aligning the orgs. I build cross-team, cross-company coalitions around hard problems and I ship the solutions myself.

I'm at home in TypeScript, Node.js, and React. I build frameworks, design systems, APIs, CLIs, and full-stack systems. I own the toolchain — build systems, CI/CD, infrastructure, databases. I write tests and hold a high bar for quality.

Right now I'm obsessed with agentic systems and the future of how we build software. Nothing pleases me more than simplicity and productivity.

Cheers!
